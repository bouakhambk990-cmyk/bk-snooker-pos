🔑 ສ້າງ GitHub Account (ຖ້າຍັງບໍ່ມີ)

📁 ສ້າງ Repository ໃໝ່

1. Login GitHub → ກົດ "+" ດ້ານຂວາເທິງ → "New repository"

2. ຕັ້ງຄ່າດັ່ງນີ້:

Fieldຄ່າທີ່ຕ້ອງໃສ່

3. ກົດ "Create repository" (ສີຂຽວ)

ຂັ້ນຕອນທີ 3
⬆️ Upload ໄຟລ໌ທັງໝົດ

ໃນ repository ທີ່ສ້າງໃໝ່ → ກົດ "uploading an existing file"

ໄຟລ໌ທີ່ຕ້ອງ upload (ທັງໝົດ 3 ອັນ):

  • 📄 index.html — ຕົວ App ຫຼັກ
  • 📋 README.md — ຄຳອະທິບາຍ
  • 📁 .github/workflows/deploy.yml — Auto-deploy
⚠️ ໄຟລ໌ .github/workflows/deploy.yml ຕ້ອງ upload ໂດຍສ້າງ folder ກ່ອນ (ເບິ່ງຂັ້ນຕອນລຸ່ມ)

ວິທີ upload deploy.yml:

1. ໃນ repo → ກົດ "Add file" → "Create new file"

2. ໃນຊ່ອງ "Name your file" ໃສ່:

.github/workflows/deploy.yml

3. Copy ເນື້ອໃນ deploy.yml ວາງໃສ່

4. ກົດ "Commit new file"

3. ໃຕ້ "Source" → ເລືອກ:

Folder/ (root)

💡 ຈະເຫັນ banner ສີຟ້າ "Your site is live at..." ເມື່ອ deploy ສຳເລັດ
