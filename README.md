## 🚀 ວິທີ Deploy ໃນ GitHub Pages

### ຂັ້ນຕອນທີ 1 — ສ້າງ Repository
1. ໄປທີ່ [github.com](https://github.com) → Login
2. ກົດ **"New repository"** (ປຸ່ມ + ດ້ານຂວາເທິງ)
3. ຕັ້ງຊື່: `bk-snooker-pos`
4. ເລືອກ **Public**
5. ກົດ **"Create repository"**

### ຂັ້ນຕອນທີ 2 — Upload ໄຟລ໌
1. ໃນ repository ໃໝ່ → ກົດ **"uploading an existing file"**
2. ລາກ/ວາງ ໄຟລ໌ທັງໝົດໃນ folder ນີ້:
   - `index.html` ← ໄຟລ໌ຫຼັກ
   - `README.md`
   - `.github/workflows/deploy.yml`
3. ກົດ **"Commit changes"**

### ຂັ້ນຕອນທີ 3 — ເປີດ GitHub Pages
1. ໄປ **Settings** → **Pages** (ຊ້າຍມື)
2. Source: **"Deploy from a branch"**
3. Branch: **main** / **root**
4. ກົດ **Save**
5. ລໍຖ້າ 1-2 ນາທີ → ໄດ້ link!
